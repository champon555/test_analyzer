# テスト結果分析ツール

`test_analyzer.py` は、CSV形式のテスト結果データを分析するためのPythonスクリプトです。現在のバージョンでは、特定の条件を満たす生徒を抽出する機能を提供しています。

## 機能

### 生徒分析 (`analyze_student` 関数)

この機能は、以下の分析を行います。

1.  **CSVデータの読み込み**: ユーザーが指定したCSVファイルを読み込みます。
2.  **データ概要の表示**: 読み込んだデータの最初の5行と、データフレームの情報を表示します。
3.  **問題列の特定**: 列名に「問題」を含む列を自動的に識別します（「氏名」列を除く）。
4.  **簡単な問題の識別**: 全体的な正答率が80%以上の問題を「簡単な問題」として識別します。
5.  **対象生徒の抽出**: 以下の条件をすべて満たす生徒を抽出します。
    * ユーザーが入力した最低点数以上の合計点を取得している。
    * 識別された「簡単な問題」のうち、いずれか一つでも間違えている。
    * これを行うことにより、点数は高いが比較的簡単な問題を間違えてしまっている生徒をリスト化することが出来ます。
6.  **結果の表示**: 抽出された生徒の氏名、合計点数、および間違えた簡単な問題のリストを表示します。

### 悪問分析 (`analyze_test` 関数)

現在、この機能は開発中です。

## 使い方

1.  スクリプトを実行します。
    ```bash
    python test_analyzer.py
    ```
2.  分析の選択を求められますので、「1」（生徒分析）または「2」（悪問分析）を入力します。
3.  「1: 生徒分析」を選択した場合、分析したいテスト結果のCSVファイル名を入力します。
4.  次に、抽出したい生徒の最低点数を入力します。

## CSVファイルのフォーマット

* テスト結果はCSV形式である必要があります。
* 生徒の名前を含む列は「氏名」という名前にしてください。（「氏名」列がない場合、生徒は行番号で識別されます。）
* 各問題の得点列は、列名に「問題」という文字列を含んでいる必要があります（例: 「問題1」など）。
* 問題の得点は、正解が `1`、不正解が `0` で記録されていることを想定しています。
